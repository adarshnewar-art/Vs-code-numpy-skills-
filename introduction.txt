>>what is numpy?
~ NumPy (Numerical Python) is one of the most important Python libraries for scientific computing, data analysis, machine learning, and AI. 
It provides a fast and efficient way to work with arrays, matrices, and mathematical operations.


>>Why we choose numpy rather than anything else?
~🚀 Fast: Written in C, making computations much faster than Python lists.
 💾 Memory Efficient: Stores data more compactly.
 ➕ Powerful Mathematical Operations: Perform calculations on entire arrays without writing loops.
 📊 Supports Multi-dimensional Arrays: Work with 1D, 2D, 3D, and higher-dimensional data.
 🤖 Foundation for AI & Data Science: Libraries like Pandas, TensorFlow, PyTorch, and Scikit-learn are built on or heavily use NumPy.

>>What is rhe one thing that shows the importance of NumPy?
~ The single most important thing NumPy provides is vectorized computation on n-dimensional arrays.
This is what makes NumPy so powerful and why almost every AI, machine learning, and data science library depends on it.

>>What is Vectorization?
~Instead of processing one element at a time using a loop, NumPy performs operations on the entire array at once.
FOR EXAMPLE TAKE THE FOLLOWING EXAMPLE:
In this program we are adding the two list a and b position wise :

WITHOUT NUMPY:
Program:                                      Output:
a = [1, 2, 3, 4, 5]                           [7, 9, 11, 13, 15]
b = [6, 7, 8, 9, 10]           

result = []

for i in range(len(a)):
    result.append(a[i] + b[i])

print(result)

WITH NUMPY:
Program:                                      Output:
import numpy as np                            [7, 9, 11, 13, 15]

a = np.array([1,2,3,4,5])
b = np.array([6,7,8,9,10])

result = a + b

print(result)

>>Why Numpy is important?
~Imagine you have 10 million numbers and you are repeating a loop for millions of times this will make the program slow because the Python interpreter handles each iteration.
NumPy performs the computation in optimized compiled code (primarily C), processing the array much more efficiently.
The result can be tens to hundreds of times faster, depending on the operation and hardware.
 
 FOR EXAMPLE:
 To make the square of every elemnts of the list:

WITHOUT numpy:                               Output:
numbers = [1,2,3,4,5]                        [1,4,9,16,25]

square = []

for x in numbers:
    square.append(x*x)

WITH numpy:                                  Output:
import numpy as np                           [1,4,9,16,25]
numbers = np.array([1,2,3,4,5])

square = numbers ** 2
